# shople

A simple Shopping Cart project submitted for The Odin Project course.



### Built With

- React + Vite + TypeScript; routing via react-router v7 (BrowserRouter, Routes, Link).
- **State**: Zustand stores (cartStore with localforage persistence, productStore for fetched items).
- **Styling**: Tailwind-style utility classes, custom UI primitives (button, badge, card, quantity, spinner).
- **Data**: FakeStore API.
- **Testing**: Vitest + React Testing Library + JSDOM.
